# KCNA5 Mutation Clustering Analysis

## Overview

This directory identifies statistically significant spatial clusters of missense
mutations along the KCNA5 protein sequence using COSMIC colorectal adenocarcinoma
(COAD) data. It also tests whether observed mutations are enriched near functionally
critical residues (within 5.5 Å of the selectivity filter) beyond random expectation,
and visualises the mutation landscape as a lollipop plot annotated with protein domains
and cluster regions.

## Directory structure

```
Clustering/
├── data/
│   └── cosmic_LI_AC_filtered.tsv       # COSMIC COAD, all columns, K+ channel genes only
│                                        # (generated by filter_gene.sh in dNdS/)
├── results/
│   └── KCNA5_clusters.csv              # NMC clustering output: significant clusters
│
├── nmc.R                               # Spatial mutation clustering via iPAC::nmc()
├── lollipop_plot.ipynb                 # Lollipop plot with domain annotation and clusters
└── mutation_simulation.ipynb           # Permutation + binomial test for selectivity
                                        # filter proximity enrichment
```

## Data

`cosmic_LI_AC_filtered.tsv` is produced by `filter_gene.sh` in `dNdS/` and is
shared between both directories. It contains all COSMIC columns, filtered to
potassium channel genes expressed in large intestine.

Raw COSMIC data is not included due to licensing restrictions — see `dNdS/README.md`
for download instructions.

## Reproduction

### Step 1 — Spatial clustering (R)

```r
source("nmc.R")
```

This script:
- Reads `data/cosmic_LI_AC_filtered.tsv` and filters to KCNA5 missense variants
- Deduplicates by sample and mutation ID
- Parses amino acid positions from `MUTATION_AA` (e.g. `p.R234H` → position 234)
- Builds a binary mutation matrix (samples × 613 residue positions)
- Runs `iPAC::nmc()` with Bonferroni correction (α = 0.05)
- Writes significant clusters to `results/KCNA5_clusters.csv`

Output columns in `KCNA5_clusters.csv`:
- `cluster_size`: number of residues spanned by the cluster
- `start` / `end`: residue positions defining the cluster
- `number`: number of mutations within the cluster
- `p_value`: Bonferroni-corrected p-value

### Step 2 — Lollipop plot (Python)

Open `lollipop_plot.ipynb`.

This notebook:
- Loads `data/cosmic_LI_AC_filtered.tsv` and parses KCNA5 missense mutations
- Counts mutations per residue position
- Plots a lollipop chart (residue position vs. count) with:
  - Protein domain annotation (T1: 1–120; S0–S5 transmembrane helices; loop helix)
  - Cluster density heatmap bar derived from `results/KCNA5_clusters.csv`
- Domain boundaries used:

| Domain       | Residues  |
|--------------|-----------|
| Disordered   | 1–120     |
| T1           | 121–216   |
| S0           | 231–240   |
| S1           | 247–270   |
| Disordered   | 282–321   |
| S2           | 323–344   |
| S3           | 349–376   |
| S4           | 392–416   |
| S5           | 433–456   |
| Loop helix   | 468–479   |

### Step 3 — Selectivity filter proximity enrichment (Python)

Open `mutation_simulation.ipynb`.

Tests whether KCNA5 missense mutations are enriched at residues within 5.5 Å of the
selectivity filter beyond random expectation. Two simulations are run:

| Simulation | n mutations | Search space        | Target sites                          |
|------------|-------------|---------------------|---------------------------------------|
| Whole protein | 103      | Residues 1–613      | 472–488, 469, 494, 501, 505 (21 sites) |
| T1 region  | 21          | Residues 449–525    | Same 21 target sites                  |

Method: Monte Carlo simulation (100,000 trials) + exact binomial test.  
Output: probability distribution plot with observed count marked.
