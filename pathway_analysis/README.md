# Transcriptomic Pathway Analysis of KCNA5 in TCGA-COAD

## Overview

This directory contains the R-based transcriptomic analysis pipeline investigating
KCNA5 expression and mutation in colorectal adenocarcinoma (COAD) using TCGA data.
Analyses include PROGENY pathway scoring, DESeq2 differential expression, and GSEA,
across three comparisons: tumour vs. normal, KCNA5 T1 domain mutants vs. wildtype,
and KCNA5 non-T1 mutants vs. wildtype.

## Data

Raw expression data is not included due to file size and TCGA data use terms.

Download from cBioPortal:
1. Go to https://www.cbioportal.org/study/summary?id=coadread_tcga_pan_can_atlas_2018
2. Download via GDC Hub: `TCGA-COAD.star_tpm.tsv` and `TCGA-COAD.star_counts.tsv`
3. Place both files in `data/raw/`

Note: GDC files are log2(x+1) transformed — `data_cleaning.R` back-transforms them
before use.

`KCNA5_mutations.txt` was downloaded from cBioPortal:
1. Go to https://www.cbioportal.org and select **Bowel** under cancer type
2. Select **Colorectal Cancer Pan Atlas (TCGA)**
3. Click **Query by Gene** and enter `KCNA5`
4. Once results load, go to the **Download** tab
5. Under **Mutations**, download as TSV format
6. Place the file in `data/raw/KCNA5_mutations.txt`

## Reproduction

Run scripts in order. All scripts should be run from the `pathway_analysis/` directory.

### Step 1 — Data cleaning (`data_cleaning.R`)

```r
source("scripts/data_cleaning.R")
```

- Back-transforms log2(x+1) TPM and counts matrices
- Converts Ensembl IDs to Hugo symbols via `org.Hs.eg.db`
- Removes unmapped and duplicate genes
- Identifies and saves tumour and normal sample lists
- Output: `data/cleaned/TCGA-COAD_tpm_cleaned.rds`,
  `data/cleaned/TCGA-COAD_counts_cleaned.rds`,
  `data/cleaned/tumour_samples.rds`, `data/cleaned/normal_samples.rds`

### Step 2 — PROGENY pathway scoring (`progeny.R`)

```r
source("scripts/progeny.R")
```

- Input: `data/cleaned/TCGA-COAD_tpm_cleaned.rds`
- Runs PROGENy on the TPM matrix (14 pathways, top 500 genes, Human)
- Correlates KCNA5 and FZD6 expression with each pathway score (Pearson)
- Output: `results/progeny/progeny_scores.rds`, `results/progeny/*.png`

### Step 3 — DESeq2: tumour vs. normal (`deseq2.R`)

```r
source("scripts/deseq2.R")
```

- Input: `data/cleaned/TCGA-COAD_counts_cleaned.rds`,
  `data/cleaned/tumour_samples.rds`, `data/cleaned/normal_samples.rds`
- Filters to genes with counts ≥ 10 in ≥ 10 samples
- Runs DESeq2 (reference level: Normal)
- Produces volcano plot highlighting KCNA5 and FZD6
- Output: `results/deseq2/DESeq2_tumour_vs_normal.csv`,
  `results/deseq2/volcano_tumour_vs_normal.pdf`, `results/deseq2/dds.rds`

### Step 4 — DESeq2: KCNA5 mutants vs. WT (`deseq2_mutations.R`)

```r
source("scripts/deseq2_mutations.R")
```

- Input: `data/cleaned/TCGA-COAD_counts_cleaned.rds`,
  `data/cleaned/tumour_samples.rds`, `data/raw/KCNA5_mutations.txt`
- Runs two contrasts within tumour samples only:
  - All KCNA5 mutants vs. WT
  - T1 domain mutants (residues 128–216) vs. WT
- Output: `results/deseq2/DESeq2_KCNA5_T1Mutant_vs_WT.csv`,
  `results/deseq2/DESeq2_KCNA5_nonT1Mutant_vs_WT.csv`,
  `results/deseq2/dds_T1mutant_vs_WT.rds`,
  `results/deseq2/dds_allMutant_vs_WT.rds`,
  `data/cleaned/wt_tumour_samples.rds`

### Step 5 — GSEA: tumour vs. normal (`gsea.R`)

```r
source("scripts/gsea.R")
```

- Input: `results/deseq2/DESeq2_tumour_vs_normal.csv`
- Builds a ranked gene list using sign(log2FC) × −log10(p-value), with zero
  p-values replaced by `.Machine$double.xmin` to avoid infinite ranks
- Runs `fgsea` (seed = 42) on Hallmark and GO:BP gene sets from MSigDB
- Filters significant pathways (padj < 0.05) and plots NES bar chart
- Output: `results/gsea/GSEA_hallmark.pdf`

### Step 6 — GSEA: T1 mutants vs. WT (`gsea_T1mutant.R`)

```r
source("scripts/gsea_T1mutant.R")
```

- Input: `results/deseq2/DESeq2_KCNA5_T1Mutant_vs_WT.csv`
- Builds ranked gene list using same method as Step 5
- Runs `fgsea` (seed = 42) on Hallmark gene sets from MSigDB
- Filters significant pathways (padj < 0.05) and plots NES bar chart
- Output: `results/gsea/T1mutant/GSEA_hallmark.pdf`,
  `results/gsea/T1mutant/GSEA_hallmark_all.csv`,
  `results/gsea/T1mutant/GSEA_hallmark_significant.csv`

### Step 7 — GSEA: non-T1 mutants vs. WT (`gsea_nonT1mutant.R`)

```r
source("scripts/gsea_nonT1mutant.R")
```

- Input: `results/deseq2/DESeq2_KCNA5_nonT1Mutant_vs_WT.csv`
- Builds ranked gene list using same method as Step 5
- Runs `fgsea` (seed = 42) on Hallmark gene sets from MSigDB
- Filters significant pathways (padj < 0.05) and plots NES bar chart
- Output: `results/gsea/nonT1mutant/GSEA_hallmark.pdf`,
  `results/gsea/nonT1mutant/GSEA_hallmark_all.csv`,
  `results/gsea/nonT1mutant/GSEA_hallmark_significant.csv`

## Dependencies

R version 4.4.3

Bioconductor packages:
```r
BiocManager::install(c("DESeq2", "progeny", "fgsea", "org.Hs.eg.db", "AnnotationDbi"))
```

CRAN packages:
```r
install.packages(c("tidyverse", "tibble", "ggplot2", "ggrepel", "msigdbr"))
```

Note: use `dplyr::select()` and `dplyr::first()` explicitly throughout to avoid
namespace conflicts with other packages.

A full session info snapshot is saved in `scripts/sessionInfo.txt`.
